CCR Login Import

Import browser login state only into a local CCR job. The URL must point to localhost or 127.0.0.1.

  1. 1 Paste the CCR import URL Create the job in CCR, then paste the copied URL here.
  2. 2 Review requested domains CCR tells the extension exactly which domains it needs.
  3. 3 Import cookies and localStorage Only the selected domains are read and sent back to the local CCR job.
