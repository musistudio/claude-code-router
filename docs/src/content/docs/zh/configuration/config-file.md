## 默认位置

- macOS：`~/Library/Application Support/claude-code-router/config.json`
- Windows：`%APPDATA%/claude-code-router/config.json`
- Linux：`~/.config/claude-code-router/config.json`

## 生效方式

直接编辑配置文件后，需要重启 CCR。
