## 能做什么

Bot 能把 Agent 消息转发到 IM 平台，也可以在桌面空闲后把任务接力到手机上继续看。

## 常见模式

- **转发 Agent 消息**：把消息同步到 IM。
- **接力**：桌面空闲后，把交互接力到 IM。

## 平台页面

Slack、Discord、Telegram、LINE、微信、企业微信、飞书和钉钉都有独立页面。
