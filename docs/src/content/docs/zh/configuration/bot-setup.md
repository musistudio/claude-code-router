## 操作流程

1. 打开 **Bot 管理**，点击 **添加 Bot**。
2. 选择平台并填写 Token、Secret、Signing Secret、Robot Code 或 OAuth 信息。
3. 保存 Bot。
4. 打开目标 Agent配置，开启 **Bot**。
5. 选择 **转发 Agent 消息** 或 **接力**，并重新从 CCR 打开 Agent。

## 验证方式

从 CCR 打开 Agent 后发一条测试消息，确认请求日志中有 Bot 相关记录，IM 端也能收到消息。
