## 添加入口

点击 **Add custom MCP** 后选择 transport。

## Transport

- **stdio**：本地命令行工具。
- **streamable-http / sse**：远程 MCP 服务。
- **Discover tools**：读取 MCP server 暴露的工具。

## 验证建议

高风险或响应慢的 MCP 工具建议先在独立配置中验证。
